Trabalhando com Listas

Listas Ordenadas

  1. Acordar
  2. boot
  3. Tomar Café
  4. Escovar os dentes

Para fazer essas litas, basta adicionar tag OL e logo a baixo a tag li e escrever o testo. Automaticamente já virá para a página ordenado. Você pode também modificar como isso vai aparecer se é em Numeros, letras (maiusculas ou minusculas) ou em algarismos romanos usando a tag type dentro da tag OL

Listas Não Ordenadas

  • Pão
  • Leite
  • Bolacha
  • Chocolate

Essas listas não precisão de uma ordem específica. Para fazer isso basta abrir a tag UL e dentro dela novamente e tag LI. Da também pra alterar o estilo como nas listas ordenadas, porém só se pode usar disc, circle ou square
