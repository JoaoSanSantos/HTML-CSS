Usando Links Externos

Você pode acessar o meu repositório público no GitHub e você pode acessar minha página de vendas.

O primeiro link levará você a um site externo, porém não abrirá uma nova aba. Já no segundo link ele abrirá uma nova aba para um link externo. Para fazer a primeira opção é colocar o link dentro da tag A e a segunda opção é após colocar o link na tag A adicione logo após as " (aspas) do link a tag target="_blank"... Dessa forma ira abrir uma nova aba para o link externo. Lembre-se: logo após a tag target dve se adicionar rel="external" indicando uma página externa.

Primeiro você deve criar um novo arquivo abaixo do index. Para usar link interno é usado a mesma tag. dentro do " " você vai precionar ctrl+espaço e vai mostrar as pastas que você criou. Deve-se adicionar também a rel="next" e na página 2 a rel="prev"


Você também pode acessar a Página 3

Caso o link interno esteja dentro de uma pasta, basta precionar ctrl+espaço e aparecerá todos os arquivos e pastas e procurar pela pasta que você criou e pelo novo arquivo.

Para voltar de uma página que esta em outra pasta, as teclas ctrl+epaço não vai funcionar. Eentão basta digitar ../ e aparecerá a página da outra pasta para você adicionar e ao adicionar ./ significa para voltar dentro da mesma pasta.
